# Initial Node.js project configuration

To save me time when I decide to create an API.

## Installation

Open your terminal and type:

```bash
$ npm install
```

## Run

### For development:

```bash
$ npm run start:dev
```

### For build:

```bash
$ npm run build
```

### Tests:

```bash
$ npm run vitest
```

### Test coverage:

```bash
$ npm run coverage
```
